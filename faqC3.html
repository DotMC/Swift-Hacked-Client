How to toggle the gui?

Toggling Gui

To toggle the gui simple press the Left-Shift button to toggle the tabbed gui and if you want to toggle the main gui press Left-Control.

(Keep in mind that the tabbed gui is only for quick mod access not for using as a the main gui because we have removed a lot mods to make it smaller and easier to navigate)
